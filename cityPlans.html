What has the MTA done so far?

The proposal for a this new line running through the bald spot that currently exist in queens was presented all the way back in 1996.
Originally called the Triboro RX, it was then proposed again in 2012, then a few more times up until the present day.
Currently named the Interborough Express, it would run over already existing track, meaning no need to build new tracks!
Beyond that, the MTA has been exceptionally secretive about the development of the track.
